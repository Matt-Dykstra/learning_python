# Python Object Oriented Programming Examples

### Code:

```python
class Person:
    pass  # An empty block

p = Person()
print(p)
```

###### Output:

```
$ python oop_simplestclass.py
<__main__.Person instance at 0x10171f518>
```

### Code
 
```python
class Person:
    def say_hi(self):
        print('Hello, how are you?')

p = Person()
p.say_hi()
# The previous 2 lines can also be written as
# Person().say_hi()
```

###### Output:

```
$ python oop_method.py
Hello, how are you?
```
